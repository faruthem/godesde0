# Goroutines

Paquete para aprender concurrencia en Go.

## Contenido

### goroutines.go
- **`goroutine`**: Funciones que se ejecutan concurrentemente
- **`channels`**: Canales para comunicación entre goroutines
- **`time.Sleep`**: Pausa en la ejecución
- **`strings.Split` y `strings.Repeat`**: Manipulación de strings
