# 😃 Contribuir

¡Gracias por tu interés en contribuir!

## 📋 Guías de Contribución

### ¿Cómo contribuir?

1. Haz un Fork del repositorio
2. Crea una rama (`git checkout -b feature/nueva-funcionalidad`)
3. Haz tus cambios y commitea (`git commit -m 'Añadir nueva funcionalidad'`)
4. Push a la rama (`git push origin feature/nueva-funcionalidad`)
5. Abre un Pull Request

### Estándares de Código

- Usa `go fmt` antes de hacer commit
- naming convention: snake_case para archivos, PascalCase para funciones
- Añade tests para nuevas funciones
- Documenta las funciones con comentarios

### Commits

Seguimos [Conventional Commits](https://www.conventionalcommits.org/):

```
feat: añadir nueva funcionalidad
fix: corregir un bug
docs: actualizar documentación
test: añadir o corregir tests
refactor: reorganizar código
```

## 🤔 ¿Dudas?

Abre un issue para discutir cambios mayores antes de hacer PR.
